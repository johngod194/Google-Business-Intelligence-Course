# Cyclistic-Summary
This repository summarises the **Cyclistic Business Intelligence Capstone** from the Google BI course on Coursera. The case focuses on analysing **Focus on understanding how their customers are using their bikes** and building a BI deliverable that helps stakeholders understand **identifying customer demand at different station locations** and how performance changes over time.

The project is split into three case studies:
- [**Case 1 - Requirements & Planning:**](https://github.com/johngod194/Google-Business-Intelligence-Course/blob/main/Projects/Cyclistic%20Business%20Intelligence%20Project/Documents%20Required/Project%20Requirements%20Document%20-%20Cyclistic%20.pdf) Define stakeholder needs and success criteria by completing the Stakeholder Requirements, Project Requirements, and Strategy documents.
- [**Case 2 - Data collection and Data Cleaning:**](https://github.com/johngod194/Google-Business-Intelligence-Course/blob/7414208eb7774344f0a50f71175f22028990a2e2/Projects/Cyclistic%20Business%20Intelligence%20Project/Case%20Scenario%20Overview/Cyclistic%20-%20Case%202%20Overview.pdf) Collected and processed data from three data sets:
  - NYC Citi Bike Trips, Census Bureau US Boundaries - [Dataset link](https://console.cloud.google.com/marketplace/product/united-states-census-bureau/us-geographic-boundaries)
  - GSOD from the National Oceanic and Atmospheric Administration - [Dataset link](https://console.cloud.google.com/marketplace/details/noaa-public/gsod)
  - Zip Code spreadsheet - [Dataset link](https://docs.google.com/spreadsheets/d/1IIbH-GM3tdmM5tl56PHhqI7xxCzqaBCU0ylItxk_sy0/template/preview#gid=806359255)
- [**Case 3 - Dashboard Development:**](https://github.com/johngod194/Google-Business-Intelligence-Course/blob/7414208eb7774344f0a50f71175f22028990a2e2/Projects/Cyclistic%20Business%20Intelligence%20Project/Case%20Scenario%20Overview/Cyclistic%20-%20Case%203%20Overview.pdf) Design and build a Tableau dashboard to visualise Summer Trends, Seasonality and Top Trips using Cyclistic matching stakeholder expectations.
